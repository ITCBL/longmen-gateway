# longmen-gateway 龙门关（网关）

## 基本信息
- 名称：龙门关（网关）
- 龙门关（Lóngmén Guān）："龙门"代表着进出通道，以及守护者的象征。
- 开发语言：go
- web框架：gin
- 数据库：mysql
- 缓存：redis
- 服务：网关代理服务 + 管理服务

## 运行
- 执行`git pull` 拉取最新代码
- 执行`go mod tidy` 下载依赖包
- 修改配置文件 `config/config-dev.yaml` 配置文件
- 执行`go run main.go` 启动程序
